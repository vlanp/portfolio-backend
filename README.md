## About The Project

This is a REST API developed with Node.js and Express that powers my portfolio web application. Data is stored in a MongoDB database and files are managed through Cloudinary. Additionally, project documentation is retrieved from GitHub repositories using the GitHub API, then processed and formatted using Remark and Rehype plugins. Schema validation is handled with Zod to ensure type safety.
